# Exercício de Dicionário em Java

Este projeto contém dois exercícios propostos pela professora:

1. **Exercício principal** (`Main.java`)
   - Criação e manipulação de um dicionário (`Map`) com estados e populações.
   - Atualização de valores, adição de novos estados, ordenação, cálculo de soma/média, remoção de elementos e verificação se o dicionário está vazio.

2. **Desafio do dado** (`Desafio.java`)
   - Simula 100 lançamentos de um dado.
   - Armazena os resultados e exibe quantas vezes cada valor foi obtido.
